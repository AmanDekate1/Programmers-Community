## Given a binary tree, find the Mirror tree of it.

Input : Root of below tree
            1
          /   \
         2      3
        / \
       4   5 
Inorder of given tree: 1 2 3 4 5

Output : Root of below tree
            1
          /   \
         3      2
               /  \
              5    4
Inorder of mirror tree: 1 3 2 5 4
